# Last Minute Life Saver

## Brand Identity

### Project Name
Last Minute Life Saver

### Tagline
AI-powered deadline rescue assistant.

### Mission
Help users complete their work before deadlines using intelligent AI planning, smart scheduling, proactive reminders, and emergency rescue planning.

### Vision
Reduce stress and missed deadlines by making AI act like a personal productivity coach.

### Target Users
- Students
- Professionals
- Freelancers
- Developers
- Anyone who struggles with deadlines
## Brand Personality

- Calm

## Core Features

- AI Task Breakdown
- Smart Task Prioritization
- Automatic Task Scheduling
- Google Calendar Integration
- Smart Rescheduling
- Panic Mode
- Firebase Cloud Messaging (FCM) Notifications
- AI Productivity Insights
### AI Task Breakdown

Gemini divides large tasks into smaller actionable subtasks.

### Smart Prioritization

AI ranks tasks based on urgency and importance.

### Smart Scheduling

AI recommends the best time slots.

### Google Calendar Sync

Automatically creates calendar events.

### Smart Rescheduling

If a task is missed, AI prepares a revised schedule.

### Firebase Cloud Messaging (FCM)

Sends:
- Deadline reminders
- Panic Mode alerts
- Daily task reminders
- AI productivity suggestions

### Panic Mode

Creates an emergency recovery plan when deadlines are near.
## Notification Types (FCM)

1. Morning Planning Reminder
   "Good morning! You have 4 tasks today."

2. Upcoming Deadline
   "Your DSA assignment is due in 6 hours."

3. Panic Mode Alert
   "Deadline is very close. AI has prepared a recovery plan."

4. Missed Task
   "You missed one task. Your schedule has been updated."

5. Daily Summary
   "Great work! You completed 6 out of 7 tasks today."
   ## AI Agents

Planning Agent
Breaks tasks into subtasks.

Priority Agent
Ranks tasks.

Scheduling Agent
Suggests the best schedule.

Recovery Agent
Creates a revised plan if work is missed.

Panic Agent
Generates an emergency completion strategy.
## Why Last Minute Life Saver?

Unlike normal to-do apps, our application thinks before reminding.

Instead of sending notifications only, it:
- Understands the task.
- Creates a plan.
- Schedules work.
- Adjusts the schedule if needed.
- Sends intelligent reminders.
- Helps users recover before deadlines.
